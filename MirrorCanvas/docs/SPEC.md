# MirrorCanvas (镜像画板) - Creative Specification

## 1. 概念与愿景

### 背景阐述
对称宇宙是对"对称"这一宇宙基本法则的艺术探索。从雪花到星系，从分子到建筑，对称性贯穿自然与人类文明。本作品将数学的对称之美转化为创作的工具，让玩家在绘制过程中体验从简单镜像到复杂万花筒的视觉奇观，感受数学与艺术的完美融合。

### 创作目标
- 展示对称性的数学之美
- 从简单到复杂的对称形式递进
- 让玩家成为对称艺术的创作者
- 探索数学规律与艺术表达的边界

### 艺术方向
以"生长"为叙事主线，玩家的每一笔都在对称法则的约束下绽放出意想不到的图案。从最初的单轴镜像，到四方连续、旋转对称，最终形成完整的曼陀罗或万花筒图案。对称性既是限制，也是创造的源泉。

## 2. 设计语言

### 美学基础
- **核心风格**: 几何对称美学，曼陀罗艺术
- **质感追求**: 线条流畅，色彩和谐
- **色彩哲学**: 渐变色彩，强调对称中心

### 色彩系统
| 类型 | 色值 | 用途 |
|------|------|------|
| 深紫 | #4a148c | 主色调 |
| 金黄 | #ffd54f | 强调色 |
| 青绿 | #00897b | 辅助色 |
| 粉红 | #f06292 | 点缀色 |
| 深蓝 | #1a237e | 背景色 |
| 白色 | #ffffff | 线条色 |

### 字体选择
- **主标题**: "Orbitron" (Google Fonts) - 几何未来感
- **辅助文字**: "Space Mono" - 等宽技术感

### 空间系统
- 圆形画布，强调中心对称
- 对称轴以虚线显示
- 边缘渐隐处理

### 动效哲学
- 绘制时实时镜像显示
- 对称轴切换时平滑过渡
- 完成作品时整体脉动效果

## 4. 功能与交互

### 核心功能

#### 4.1 对称绘制系统
- **镜像对称**: 单轴镜像，左右对称
- **四方对称**: 双轴镜像，上下左右对称
- **六角对称**: 60度旋转对称，六瓣花形
- **八边对称**: 45度旋转对称，曼陀罗形

#### 4.2 绘制工具
- **画笔**: 自由绘制线条
- **颜色**: 色板选择 + 自定义
- **粗细**: 1px - 20px 可调
- **清除**: 清空画布重新开始

#### 4.3 作品保存
- **导出**: PNG 格式下载
- **分享**: 生成分享链接（可选）

### 交互细节
| 交互 | 视觉反馈 |
|------|----------|
| 鼠标按下 | 开始绘制，实时镜像 |
| 鼠标移动 | 线条跟随，对称复制 |
| 切换模式 | 对称轴动画过渡 |
| 清除画布 | 渐隐消失效果 |

### 边界情况
- 绘制超出画布边界时裁剪
- 快速移动时线条平滑插值
- 多点触控时只响应第一个点

## 5. 组件清单

### 5.1 对称画布组件 (SymmetryCanvas)
- **形状**: 圆形
- **对称轴**: 可配置数量
- **绘制**: 实时镜像渲染

### 5.2 对称模式选择器 (SymmetrySelector)
- **选项**: 镜像、四方、六角、八边
- **样式**: 图标按钮
- **状态**: 激活、非激活

### 5.3 颜色选择器 (ColorPicker)
- **类型**: 色板 + 自定义
- **预设**: 8 种预设颜色
- **自定义**: HSL 滑块

### 5.4 画笔粗细滑块 (BrushSlider)
- **范围**: 1px - 20px
- **样式**: 滑块 + 数值显示
- **预览**: 实时预览粗细

### 5.5 工具栏 (Toolbar)
- **功能**: 颜色、粗细、清除、保存
- **位置**: 底部固定
- **样式**: 极简图标

## 6. 技术实现

### 框架选择
- **渲染**: HTML5 Canvas 2D
- **对称计算**: 数学变换矩阵
- **动画**: requestAnimationFrame

### 性能目标
- 60fps 流畅绘制
- 对称轴数量: 2-8
- 线条平滑度: 高

- Chrome 80+
- Firefox 75+
- Safari 13+
- iOS Safari 13+
- Android Chrome 80+
