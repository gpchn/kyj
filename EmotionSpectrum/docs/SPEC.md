# EmotionSpectrum (情绪光谱) - Creative Specification

## 1. 概念与愿景

### 背景阐述
情绪光谱是一次将无形的声音转化为有形的色彩的艺术实验。灵感来源于抽象表现主义大师杰克逊·波洛克的"行动绘画"理念——艺术创作本身就是一种情感的宣泄与表达。本作品将环境声音（或鼠标运动）实时转化为流动的色彩，创造出一幅永不停歇的"流体画"，让观者见证声音与色彩的奇妙共鸣。

### 创作目标
- 将声音信号实时转化为视觉艺术
- 探索声音强度与色彩温度的映射关系
- 创造沉浸式的视听通感体验
- 让每个人都能"看见"自己的声音

### 艺术方向
以"流动"为核心美学，模拟颜料在水中扩散、交融的质感。画面不追求静态的完美构图，而是强调动态的韵律美——声音的起伏即是画面的呼吸，色彩的流转即是情绪的波动。

## 2. 设计语言

### 美学基础
- **核心风格**: 抽象表现主义，流体艺术
- **质感追求**: 颜料流动、水彩晕染的有机质感
- **色彩哲学**: 声音强度映射色彩温度

### 色彩系统
| 类型 | 色值 | 情绪映射 |
|------|------|----------|
| 热情红 | #ff4757 | 高强度、激动 |
| 活力橙 | #ffa502 | 较高强度、热情 |
| 温暖黄 | #fffa65 | 中高强度、愉悦 |
| 平静蓝 | #70a1ff | 中低强度、宁静 |
| 深邃紫 | #5352ed | 低强度、沉思 |
| 神秘靛 | #3742fa | 极低强度、内省 |

### 字体选择
- **主标题**: "Playfair Display" (Google Fonts) - 优雅衬线
- **辅助文字**: "Lato" - 现代无衬线

### 空间系统
- 全屏沉浸式画布
- 无边框设计，色彩延伸至屏幕边缘
- UI 元素悬浮于画面之上

### 动效哲学
- 流体运动采用粒子系统模拟
- 色彩过渡平滑渐变，无突变
- 整体画面呈现"呼吸"般的节奏感

## 4. 功能与交互

### 核心功能

#### 4.1 声音可视化系统
- **输入源**: 麦克风环境音 / 鼠标运动速度
- **处理**: Web Audio API 实时分析音频频谱
- **映射**: 音量 → 流速，频率 → 色相

#### 4.2 流体渲染系统
- **粒子数量**: 动态调整，最高 1000 粒子
- **运动模式**: 基于噪声场的有机流动
- **色彩混合**: 粒子重叠时颜色叠加

#### 4.3 交互模式切换
- **声音模式**: 使用麦克风捕获环境音
- **鼠标模式**: 使用鼠标速度作为输入
- **切换方式**: 点击模式切换按钮

### 交互细节
| 输入 | 视觉反馈 |
|------|----------|
| 大声/快速移动 | 暖色调、高速流动、大粒子 |
| 小声/慢速移动 | 冷色调、缓慢流动、小粒子 |
| 静止/静音 | 画面逐渐平静，色彩沉淀 |

### 边界情况
- 麦克风权限被拒绝时自动切换到鼠标模式
- 长时间无输入时画面进入"休眠"状态
- 浏览器不支持 Web Audio 时降级处理

## 5. 组件清单

### 5.1 流体粒子组件 (FluidParticle)
- **状态**: 活跃、衰减、消亡
- **尺寸**: 2px - 20px 动态
- **透明度**: 0.1 - 0.8 渐变

### 5.2 色彩映射器 (ColorMapper)
- **输入**: 音量值 (0-1)
- **输出**: HSL 色彩值
- **算法**: 线性插值 + 平滑过渡

### 5.3 音量指示器 (VolumeIndicator)
- **样式**: 圆形波纹动画
- **颜色**: 随当前音量变化
- **位置**: 右下角固定

### 5.4 模式切换器 (ModeSwitcher)
- **选项**: 声音模式 / 鼠标模式
- **样式**: 极简图标按钮
- **位置**: 左下角固定

## 6. 技术实现

### 框架选择
- **渲染**: HTML5 Canvas 2D
- **音频**: Web Audio API
- **动画**: requestAnimationFrame

### 性能目标
- 60fps 流畅渲染
- 粒子数量上限: 1000
- 音频采样率: 44100Hz

### 兼容性

- Firefox 75+
- Safari 14+ (Web Audio 限制)
- iOS Safari 14+
- Android Chrome 80+
