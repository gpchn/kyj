# FractalStudio (分形绘制器) - Creative Specification

## 1. 概念与愿景

### 背景阐述
生命之线是对"线"这一美术基本元素的深度探索。线，是绘画的骨架，是情感的轨迹，是生命的隐喻。本作品受现代主义大师保罗·克利"用一根线条去散步"的理念启发，将线条从静态的描绘工具转化为动态的生命体，让观者见证线条如何从混沌中生长出秩序，从抽象中孕育出具象。

### 创作目标
- 展现"线"作为美术基本元素的韵律美
- 探索从抽象到具象的形态转化过程
- 创造有机的线条生长动画效果
- 让观者参与线条的生命历程

### 艺术方向
以"生长"为核心美学，线条不是被绘制，而是被"培育"。画面初始呈现杂乱的线条网络，随着观者的点击与引导，线条开始有目的地生长、缠绕、交织，最终形成特定的形态——器官、花朵、星座，象征生命的多样性与统一性。

## 2. 设计语言

### 美学基础
- **核心风格**: 现代主义构成感，极简几何美学
- **质感追求**: 线条的流动性、圆点的节奏感
- **色彩哲学**: 莫兰迪色系的低饱和度优雅

### 色彩系统
| 类型 | 色值 | 用途 |
|------|------|------|
| 莫兰迪灰 | #8b8589 | 主要线条 |
| 莫兰迪粉 | #c4a7a7 | 强调线条 |
| 莫兰迪蓝 | #7a8b8b | 辅助线条 |
| 莫兰迪绿 | #8a9a8a | 生长提示 |
| 莫兰迪黄 | #c9b896 | 高亮点 |
| 背景米 | #f5f3ef | 画布底色 |

### 字体选择
- **主标题**: "Cormorant Infant" (Google Fonts) - 优雅纤细
- **辅助文字**: "Quicksand" - 圆润现代

### 空间系统
- 留白为主，线条稀疏分布
- 黄金分割点作为生长起点
- 动态平衡的构图原则

### 动效哲学
- 线条生长采用贝塞尔曲线插值
- 生长速度模拟植物生长的节奏
- 分支产生具有随机性但遵循整体方向

## 4. 功能与交互

### 核心功能

#### 4.1 线条生长系统
- **初始状态**: 画布上散落杂乱的线条网络
- **触发**: 鼠标点击
- **效果**: 点击处线条开始向目标形态生长
- **过程**: 线条像藤蔓一样缠绕、延伸

#### 4.2 形态引导系统
- **预设形态**: 器官(心脏)、花朵(玫瑰)、星座(北斗)
- **自由模式**: 线条跟随鼠标轨迹生长
- **切换**: 点击形态选择器切换

#### 4.3 生长控制
- **加速**: 按住鼠标加速生长
- **暂停**: 空格键暂停/继续
- **重置**: 双击清空重新开始

### 交互细节
| 交互 | 视觉反馈 |
|------|----------|
| 点击空白处 | 显示生长起点光晕 |
| 线条生长中 | 末端有轻微发光 |
| 形态完成 | 整体轻微脉动 |
| 鼠标悬停线条 | 线条微微变亮 |

### 边界情况
- 线条生长到边界自动转向
- 多次点击叠加生长效果
- 形态完成后续点击可添加装饰

## 5. 组件清单

### 5.1 线条组件 (Line)
- **状态**: 静止、生长中、完成
- **属性**: 起点、终点、曲率、粗细
- **动画**: 生长进度 0-100%

### 5.2 生长点组件 (GrowthPoint)
- **功能**: 线条生长的引导点
- **样式**: 小圆点 + 光晕
- **行为**: 吸引附近线条

### 5.3 形态模板 (ShapeTemplate)
- **类型**: 器官、花朵、星座、自由
- **数据**: SVG 路径数据
- **渲染**: 转换为生长引导点序列

### 5.4 形态选择器 (ShapeSelector)
- **样式**: 图标按钮组
- **状态**: 激活、非激活、禁用
- **位置**: 底部居中

## 6. 技术实现

### 框架选择
- **渲染**: HTML5 Canvas 2D
- **路径**: SVG 路径解析
- **动画**: requestAnimationFrame

### 性能目标
- 60fps 流畅渲染
- 线条数量上限: 500
- 生长动画时长: 3-10秒
