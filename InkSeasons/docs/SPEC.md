# InkSeasons (墨韵四季) - Creative Specification

## 1. 概念与愿景

### 背景阐述
墨韵·四季是一次将中国传统水墨艺术与现代交互技术深度融合的视觉探索。本作品以宋代山水画的意境为核心，追寻"气韵生动"这一中国美学至高境界，通过数字技术重塑毛笔水墨在宣纸上的独特质感，让观者在虚拟空间中体验"墨分五色"的东方哲学。

### 创作目标
- 复现传统水墨艺术的"干、湿、浓、淡、焦"五色层次
- 通过四季轮转隐喻生命周期的自然律动
- 将"留白"哲学转化为交互设计的语言
- 让观者成为创作的参与者，而非被动观赏者

### 艺术方向
以"简、淡、静、远"为美学纲领，追求"意到笔不到"的含蓄之美。画面不追求写实的具象描绘，而是通过抽象的墨点流动，让观者自行感悟四季的神韵。

## 2. 设计语言

### 美学基础
- **核心风格**: 纯水墨风格，遵循中国传统绘画美学
- **质感追求**: 宣纸纹理背景，模拟真实的笔墨渗透效果
- **笔触特性**: 干湿浓淡分明，体现毛笔的书法韵味

### 色彩系统
| 类型 | 色值 | 用途 |
|------|------|------|
| 墨黑 | #1a1a1a | 主要墨迹 |
| 淡墨 | #4a4a4a | 远景渲染 |
| 飞白 | #f5f5f0 | 宣纸底色 |
| 桃花粉 | #ffb7c5 | 春季点缀 |
| 荷叶绿 | #2d5a27 | 夏季主色 |
| 枫叶红 | #c41e3a | 秋季高潮 |
| 雪花白 | #e8e8e8 | 冬季意象 |

### 字体选择
- **主标题**: "Ma Shan Zheng" (Google Fonts) - 楷书韵味
- **辅助文字**: "Noto Serif SC" - 宋体端庄

### 空间系统
- 画面采用中国画"散点透视"原则
- 黄金比例参考: 3:4 竖幅构图
- 留白区域不低于画面的40%

### 动效哲学
- 墨点晕染采用贝塞尔曲线路径
- 四季切换以淡入淡出为主，周期4秒
- 所有动画遵循"缓起缓收"的时间曲线

### 页面架构

├─────────────────────────────────┤
│                                 │

### 响应式策略
- 桌面端: 全屏画布，标题悬浮于左上角
- 平板端: 保持3:4画布比例，季节指示器移至底部
- 移动端: 9:16竖幅构图，支持触摸交互

## 4. 功能与交互

### 核心功能

#### 4.1 墨点生成系统
- **触发**: 鼠标移动/触摸滑动
- **效果**: 鼠标轨迹生成墨点，墨点根据当前季节呈现不同形态
- **春季**: 桃花效果 - 粉红色花瓣状墨点
- **夏季**: 荷叶效果 - 绿色晕染扩散
- **秋季**: 枫叶效果 - 红色裂纹扩散
- **冬季**: 雪花效果 - 白色结晶凝固

#### 4.2 季节切换系统
- **自动切换**: 每30秒自动过渡到下一季节
- **手动切换**: 点击季节指示器可立即跳转
- **过渡动画**: 4秒渐变，前3秒旧季节淡出，后1秒新季节显现

#### 4.3 留白重置
- **触发**: 双击空白区域
- **效果**: 画面逐渐恢复至空白宣纸状态

### 交互细节
| 交互 | 视觉反馈 |
|------|----------|
| 鼠标移入画布 | 显示淡雅的墨点光晕提示 |
| 鼠标快速移动 | 墨迹细而干 |
| 鼠标慢速移动 | 墨迹粗而湿 |
| 长时间静止 | 墨迹自然晕开 |

### 边界情况
- 多点触控时只响应第一个触点
- 浏览器失焦暂停动画以节省资源
- 触摸设备禁用hover状态

## 5. 组件清单

### 5.1 墨点组件 (InkDot)
- **状态**: 干燥、湿润、晕染、凝固
- **尺寸**: 5px - 50px 随机
- **透明度**: 0.3 - 0.9 渐变

### 5.2 季节指示器 (SeasonIndicator)
- **状态**: 激活(实心圆)、非激活(空心圆)
- **动画**: 激活时外圈呼吸动画
- **尺寸**: 12px 直径，间距16px

### 5.3 标题组件 (Title)
- **字体**: Ma Shan Zheng, 48px
- **颜色**: #1a1a1a, 透明度随季节变化
- **位置**: 左上角，距边缘40px

### 5.4 宣纸背景 (RicePaper)
- **纹理**: 细腻纤维纹理叠加
- **颜色**: #f5f5f0 底色
- **噪点**: 5% 高斯噪点模拟纸张质感

## 6. 技术实现

### 框架选择
- **渲染**: HTML5 Canvas 2D
- **动画**: requestAnimationFrame
- **音频**: Web Audio API (可选环境音)

### 性能目标
- 60fps 流畅渲染
- 单帧墨点数量上限: 500
- 内存回收阈值: 墨点年龄超过10秒自动清除

### 兼容性
- Chrome 80+
- Firefox 75+
- Safari 13+
- iOS Safari 13+
- Android Chrome 80+
